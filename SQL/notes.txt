----Operators---------

-----LIMIT CLAUSE------
Sets an upper limit on number of tuples to be returned
-> SELECT col1, col2 FROM table_name
    LIMIT number;

-----ORDER by CLAUSE------
To sort in ascending(ASC) or descending order (DESC)
-> SELECT col1, col2 FROM table_name
    ORDER BY col_name(s) ASC;

-----AGGREGATE FUNC------
Perform a calculation on a set of values and returns a single value.
-> COUNT()
-> MAX()
-> MIN()
-> SUM()
-> AVG()

---Group by CLAUSE-----
Groups rows that have the same values into summaray rows.
It collects data from multiple records and groups the result by one or maore column.
Generaly use with some aggregate func

-> SELECT col1, col2 FROM table_name
    GROUP BY col_name(s);

If we want to print a col beside aggregate function then we need to make a group by that col

--------HAVING CLAUSE--------
Similar to WHERE, but used when we want to apply any condition after grouping
Grouping is necessary for HAVING

-> SELECT col1, col2 FROM table_name
    GROUP BY col_name(s)
    HAVING condition;

----General Order-------
SELECT col(s)
FROM table_name
WHERE condition
GROUP BY col(s)
HAVING condition
ORDER BY col(s) ASC
